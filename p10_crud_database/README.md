# Tentang
Website yang Anda kunjungi ini adalah wujud nyata dari proses pembelajaran kami dalam mata kuliah Pemrograman WEB. Ini bukan sekadar Tugas Besar biasa, melainkan sebuah arena bagi kami untuk mengaplikasikan ilmu, mengatasi tantangan debugging, dan belajar berkolaborasi secara efektif. Kami adalah sebuah tim yang terdiri dari 4 individu dengan latar belakang dan keahlian yang beragam.
